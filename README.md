# AI/ML Mini Projects Collection

## Overview
This repository contains a collection of **beginner-friendly AI/ML mini projects** that I built while learning Artificial Intelligence and Machine Learning as a fresher.

The main goal of these projects was to gain practical experience by working on real-world problem statements and understanding how machine learning models and NLP techniques are applied in actual applications.

These projects helped me improve my understanding of:
- data preprocessing
- machine learning workflows
- natural language processing (NLP)
- recommendation systems
- frontend integration using Streamlit

This collection currently includes **3 mini projects**, each built as part of my learning and internship practice.

---

## Projects Included

### 1. Fake News Detection Using Machine Learning
This project classifies whether a given news article is **Fake** or **Real** using NLP and machine learning techniques.

**Main concepts used:**
- Text preprocessing
- TF-IDF Vectorization
- Logistic Regression
- Streamlit UI

**Project folder:** `fake_news_detection/`

---

### 2. AI-Powered Resume Ranker Using NLP
This project ranks multiple resumes based on how well they match a given job description.

**Main concepts used:**
- PDF text extraction
- TF-IDF Vectorization
- Cosine Similarity
- Streamlit UI

**Project folder:** `resume_ranker/`

---

### 3. Movie Recommendation System Using Machine Learning
This project recommends similar movies based on movie content such as genre and description.

**Main concepts used:**
- Content-based filtering
- TF-IDF Vectorization
- Cosine Similarity
- Streamlit UI

**Project folder:** `movie_recommender/`

---
